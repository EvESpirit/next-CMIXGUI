
# next-CMIXGUI
An updated C#/WinUI3-based refactor of the [cmix_gui project](https://github.com/moisespr123/cmix_gui).

![next-CMIXGUI](https://github.com/user-attachments/assets/a16445fb-3ffe-42c7-9612-eeb8c287e0ca)


### GUI Usage:
Download the nextCMIXGUI.rar file from this repository's releases.

>*(Optional) Verify with a free tool like File Hasher (available on the Microsoft Store) or using PowerShell's inbuilt ```Get-FileHash md5.exe``` capabilities.*

```SHA256 for the 1.0 release: 7F0DFF1447CF5F8BF8CC906BBA3166E99AAEB0858FC05BACAA024627B455D3B1```

>*(Optional) Set your desired language in the top right - English or Spanish.*

Run it, point it towards a file you want to compress/decompress, choose Compress or Extract, choose whether or not to use a dictionary (english.dic is provided), and hit start to proceed with the operation.
